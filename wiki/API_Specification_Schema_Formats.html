API Specification Schema Formats

Types

In contrast to JSON Schema, an OpenAPI document is a definition for an entire API, not just data models.

OpenAPI

A description format for REST/HTTP APIs that allows you to map your API's input/output parameters, endpoints, and authentication methods.

JSON Schema

You can use JSON Schema to describe data objects for both requests and responses. However, OpenAPI includes how those requests and responses are formatted. Similarly, you can mock API responses with JSON Schema, but you need something like OpenAPI to generate an entire mock server. More info at https://json-schema.org/understanding-json-schema/.
